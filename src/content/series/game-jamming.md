Everyone is learning, and there's so much to learn in game development.

Game jams are a great and low stakes way to learn on game development. Why?

- Get in the rhythm of shipping something.
- Find and learn key applications and tools.
- Learn your game engine by doing something active.
- Vary art styles, game type, mechanics to get a broad set of experience.

In this series, I'm going to capture my process (as I discover it!) in the hope it'll be useful to others.
